# Postfix Expression Evaluation Using Stack (C#)

This project demonstrates how to evaluate a **postfix expression** using a **stack** in C#.

## What is Postfix Expression?
In postfix notation (Reverse Polish Notation), the operator comes **after** the operands.

## Example:
382/+5-
Evaluation:
3 8 2 / + 5 - = 2

## Features
- Stack-based evaluation
- Supports operators: `+`, `-`, `*`, `/`
- Handles invalid expressions and division by zero
- Demonstrates LIFO behavior of stacks
